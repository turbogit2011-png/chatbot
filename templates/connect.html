Jak uzyskać klucze API WooCommerce?

  1. Zaloguj się do panelu WordPress → WooCommerce → Ustawienia
  2. Zakładka: Zaawansowane → REST API
  3. Kliknij Dodaj klucz, opis: "TurboSEO Engine", Uprawnienia: Odczyt/Zapis
  4. Skopiuj Consumer Key (ck_...) i Consumer Secret (cs_...)

Do aktualizacji meta SEO (Yoast/RankMath) i stron WordPress potrzebne są dane logowania WP:

WordPress → Użytkownicy → Twój profil → Hasła aplikacji

Bez klucza AI działa w trybie szablonowym (bez Claude). Pobierz klucz: console.anthropic.com
